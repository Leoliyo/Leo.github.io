# __What is Shounen?__
### Shounen is a genre of Japanese comics and animated films aimed primarily at a young male audience, typically characterized by action-filled plots.
# __Types of shounen__
### 1. Anime
### 2. Manga
### 3. Manwha
### 4. Manhua
